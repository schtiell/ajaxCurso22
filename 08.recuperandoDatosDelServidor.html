AJAX: Asynchronous Javascript And XML Propiedad: responseText

La propiedad responseText del objeto XMLHttpRequest. Esta propiedad almacena el valor devuelto por el servidor.

Normalmente se accede a la propiedad responseText cuando el objeto XMLHttpRequest informa que toda la información fue remitida por el servidor, esto ocurre cuando la propiedad readyState del objeto XMLHttpRequest almacena el valor 4.

Confeccionaremos un ejemplo para identifiar donde utilizar la propiedad responseText. El problema consiste en mostrar una lista de hipervínculos que representan los comentarios de distintas fecha. El objetivo es rescatar todos los comentarios para la fecha seleccionada por el visitante y su posterior visualización sin tener que recargar nuevamente la página.
