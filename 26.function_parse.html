AJAX: Asynchronus Javascript And XML JSON a Js: function parse( )

El método parse() es una función del objeto JSON, este método recibe un objeto determinado en sus parentesis, este parametro es un string, es decir una cadena de caracteres.

Al recibir el string el método parse() se encargará de transformarlo en un objeto. Esto permite transformar una cadena de caracteres en un objeto y acceder a sus propiedades.
