Ventajas y desventajas de AJAX

  1. Utiliza tecnologias ya existentes
  2. Soportado por la mayoria de los navegadores modernos
  3. Interactividad. El usuario no tiene que esperar a que lleguen los datos del servidor
  4. Portabilidad. No requiere de plugins
  5. Mayor velocidad. Debido a que no debe recargar todo el sitio completo
  1. Se pierde el concepto de volver a la página anterior
  2. Si se guarda en favoritos no necesariamente se guarda en la posición actual
  3. La existencia de sitios con ajax y otros sin esta, favorce el desorientarse
  4. Problemas de compatibilidad con navegadores antiguos
  5. No funciona si el usuario ha desactivado javascript en el navegador
  6. Se requiere conocimiento en todas las tecnologias utilizadas por AJAX
  7. Tiempos de carga lentos dependiendo de lacarga del servidor
