AJAX: Asynchronous Javascript And XML XMLHttpRequest object

El objeto XMLHttpRequest es un objeto fundamental para la comunicación asíncrona con el servidor, este objeto permite enviar y recibir información en formato XML y en general en cualquier formato.

Basicamente el objeto XMLHttpRequest proporciona una forma más facil de obtener información de una URL sin tener que recargar la página completa. Apesar de su nombre y como se mencionó en el anterior párrafo, el objeto XMLHttpRequest admite otros formatos además del XML incluso file y ftp.

Para crear una instancia de XMLHttpRequest, la sintaxis es la siguiente:

            var objeto = new XMLHttpRequest();
